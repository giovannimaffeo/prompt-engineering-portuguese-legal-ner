### Tarefa
Sua tarefa é gerar uma versão HTML de um texto de entrada, marcando entidades específicas relacionadas ao domínio jurídico. As entidades a serem identificadas são: 'pessoa', 'organizacao', 'tempo', 'local', 'legislacao' e 'jurisprudencia'. Use tags HTML <span> para destacar essas entidades. Cada <span> deve ter um atributo class indicando o tipo da entidade.

### Guia de Marcação de Entidades
Use <span class="pessoa"> para denotar uma pessoa.
Use <span class="organizacao"> para denotar uma organização.
Use <span class="tempo"> para denotar tempo/data.
Use <span class="local"> para denotar um local.
Use <span class="legislacao"> para denotar legislação.
Use <span class="jurisprudencia"> para denotar jurisprudência.
Deixe o texto como está se nenhuma dessas entidades for encontrada.

### Formato de Saída
Retorne apenas o texto original anotado em HTML. Não inclua explicações ou blocos de código.
